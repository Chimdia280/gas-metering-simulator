# Gas Metering Skid Simulator

Digital twin prototype of a natural gas metering system:

Warri Escravos Pipeline → Scrubber → Heater → Pressure Reduction → Coriolis Meter

## Features
- Real-time pressure & temperature simulation
- Coriolis metering
- Gas chromatograph input
- Energy calculation (HHV, Wobbe Index)
- Interactive dashboard

## Deployment
Hosted on Railway
